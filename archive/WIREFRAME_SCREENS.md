# BrewSpot 화면 목록 / 와이어프레임 초안

## 1. 문서 목적

이 문서는 비개발자도 피그마나 바이브코딩 툴에 바로 옮길 수 있도록  
MVP 기준 화면 목록과 각 화면의 핵심 UI 요소를 정리한 문서다.

## 2. 하단 탭 구조

## 3. 전체 화면 목록

## A. 온보딩 / 로그인

### A-1. 스플래시

- 앱 로고
- 앱 이름
- 자동 로그인 체크

### A-2. 온보딩

- 서비스 소개 카드 3장
- 시작하기 버튼
- 둘러보기 버튼

### A-3. 로그인

- Gmail 로그인 버튼
- Kakao 로그인 버튼
- Naver 로그인 버튼
- 이메일 로그인 버튼
- 비회원 둘러보기 버튼
- 개인정보처리방침 / 이용약관 링크

### A-4. 이메일 로그인

- 이메일 입력
- 비밀번호 입력
- 로그인 버튼
- 비밀번호 찾기
- 회원가입 이동

### A-5. 이메일 회원가입

- 이메일
- 비밀번호
- 비밀번호 확인
- 닉네임
- 약관 동의
- 가입 버튼

### A-6. 최초 가입 정보 설정

- 프로필 이미지
- 닉네임
- 관심 커피 태그 선택
- 선호 지역 선택
- 완료 버튼
