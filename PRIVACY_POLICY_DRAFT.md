# BrewSpot 개인정보처리방침(MVP 기준안)

최종 정리일: 2026-04-24

이 문서는 현재 BrewSpot MVP 구현 범위를 기준으로 정리한 개인정보처리방침 기준안이다.  
실제 공개 전에는 운영사명, 문의처, 위탁 현황, 법정 보관 항목을 최종 확인해야 한다.
고정값 확인은 `LAUNCH_VALUES_CHECKLIST.md`를 기준으로 진행한다.

## 1. 총칙

BrewSpot(이하 "회사")은 개인정보 보호법 등 관련 법령을 준수하며, 이용자의 개인정보를 서비스 제공에 필요한 범위에서만 처리한다.

## 2. 처리하는 개인정보 항목

### 회원가입 및 로그인

1. 이메일 로그인 시
- 이메일 주소
- 비밀번호
- 닉네임

2. 서비스 공통 식별 정보
- 사용자 내부 ID
- 가입/로그인 시각

### 서비스 이용 과정에서 처리되는 정보

1. 작성한 리뷰 내용, 평점, 추천 메뉴
2. 북마크한 카페 정보
3. 취향 태그, 마이페이지 표시 정보
4. 현재 위치 기반 지도 이동과 거리 표시에 필요한 기기 위치정보
5. 서비스 접속 기록, 오류 로그, 기기/앱 버전 정보가 운영 과정에서 생성될 수 있음

### 현재 MVP에서 처리하지 않는 항목

1. 프로필 이미지 업로드
2. 마케팅 수신 정보
3. 광고 식별자 기반 맞춤형 광고 정보

## 3. 개인정보의 처리 목적

1. 회원 식별, 로그인, 계정 관리
2. 리뷰, 북마크, 마이페이지 기능 제공
3. 카페 탐색 및 서비스 개인화
4. 부정 이용 방지, 운영 정책 집행, 보안 대응
5. 장애 분석, 품질 개선, 서비스 통계 확인
6. 문의 대응 및 공지 전달

## 4. 개인정보의 처리 및 보유 기간

1. 회원 정보는 탈퇴 시까지 보유한다.
2. 리뷰, 북마크, 활동 정보는 서비스 운영 목적 범위에서 회원 탈퇴 전까지 보유한다.
3. 관계 법령에 따라 보관이 필요한 정보는 해당 법정 기간 동안 별도 보관한다.
4. 탈퇴 또는 처리 목적 달성 후에는 지체 없이 파기한다.

## 5. 개인정보의 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않는다.  
다만 아래에 해당하는 경우에는 예외가 있을 수 있다.

1. 이용자가 사전에 동의한 경우
2. 법령에 따라 제공 의무가 있는 경우
3. 수사기관 등 적법한 절차에 따른 요청이 있는 경우

## 6. 개인정보 처리 위탁

현재 MVP 운영 기준으로 아래 업무를 외부 서비스에 위탁할 수 있다.

1. 인증, 데이터베이스 인프라 운영: Supabase

실제 공개 시에는 수탁업체 명칭, 위탁업무, 보유기간을 최신 상태로 공개한다.

## 7. 개인정보의 파기 절차 및 방법

1. 보유기간이 끝나거나 처리 목적이 달성된 개인정보는 지체 없이 파기한다.
2. 전자적 파일은 복구 또는 재생이 어렵도록 삭제한다.
3. 출력물은 분쇄 또는 소각한다.

## 8. 이용자의 권리와 행사 방법

이용자는 언제든지 아래 권리를 행사할 수 있다.

1. 개인정보 열람 요청
2. 개인정보 정정 요청
3. 동의 철회 요청
4. 회원 탈퇴 요청
5. 처리 정지 요청

회사는 법령에서 정한 사유가 없는 한 지체 없이 필요한 조치를 한다.

## 9. 개인정보의 안전성 확보 조치

회사는 개인정보 보호를 위해 아래 조치를 적용한다.

1. 접근 권한 최소화
2. 인증 정보 및 세션 관리
3. 전송 구간 보호
4. 관리자 권한 분리
5. 접속 및 오류 로그 관리
6. 보안 업데이트와 점검

## 10. 위치정보 처리에 관한 안내

현재 BrewSpot은 이용자가 위치 권한을 허용한 경우, 앱 내 지도에서 `내 위치` 이동과 카페 거리 표시를 위해 기기 현재 위치를 사용할 수 있다.  
현재 구현 기준으로 이 위치정보는 앱 화면 계산에만 사용되며, 서버에 저장하거나 백그라운드에서 추적하지 않는다.

향후 근처 추천, 거리순 정렬 고도화, 위치 기록, 서버 저장 같은 기능을 도입하는 경우에는 아래를 함께 반영한다.

1. 위치 권한 요청 문구 추가
2. 위치정보 관련 동의 절차 추가
3. 개인정보처리방침 개정
4. 위치기반서비스 약관 공개 여부 재검토
