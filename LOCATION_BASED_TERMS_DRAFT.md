# BrewSpot 위치정보 처리 기준 메모

최종 정리일: 2026-04-24

## 1. 현재 MVP 판단

현재 BrewSpot iOS 앱은 이용자의 현재 위치 권한을 요청하지 않으며, 개인위치정보를 수집하거나 저장하지 않는다.

현재 구현 범위:

1. 카페 좌표를 지도에 표시
2. 지역별 카페 탐색
3. 카페 상세에서 매장 좌표 노출

현재 구현 범위에 포함되지 않는 것:

1. 이용자의 현재 위치 수집
2. 근처 카페 추천
3. 거리순 정렬
4. 백그라운드 위치 추적

따라서 현재 MVP 기준으로는 개인위치정보를 직접 처리하는 위치기반서비스 단계로 보지 않고, 별도 위치기반서비스 이용약관을 앱에 공개하지 않는 방향으로 정리한다.

## 2. 향후 위치기능 도입 시 선행 조건

아래 기능 중 하나라도 추가하면 위치정보 관련 문서를 다시 공개 기준으로 전환한다.

1. 현재 위치 중심 지도 이동
2. 근처 카페 자동 추천
3. 거리순 정렬
4. 이용자 위치 기반 알림 또는 기록 기능

## 3. 향후 반영 체크리스트

위 기능을 도입할 경우 출시 전에 아래를 함께 반영해야 한다.

1. `Info.plist` 위치 권한 문구 추가
2. 앱 내 위치 동의 및 거부 흐름 추가
3. 개인정보처리방침 위치정보 항목 개정
4. 위치정보 관련 고지 문구 및 이용약관 공개 여부 재검토
5. 위치정보법상 신고 또는 추가 의무 해당 여부 재검토

## 4. 참고 메모

현재 판단은 `이용자 개인위치정보`를 처리하지 않는 현행 코드 기준이다.  
정적 카페 좌표 표시만으로는 이용자 개인위치정보 처리 단계로 보지 않았다.
