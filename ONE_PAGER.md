# BrewSpot 1페이지 요약

최종 기준일: 2026-05-01

## 프로젝트

`BrewSpot`  
내 취향에 맞는 카페를 찾고, 저장하고, 리뷰로 기록하는 iOS 커피 앱

## 지금 풀고 있는 문제

카페 정보는 많지만 사용자는 여전히 "내 취향에 맞는 카페"를 빠르게 찾기 어렵다.  
현재 MVP는 이 문제를 `지도 + 카페 상세 + 리뷰 + 저장 + 마이페이지 기록` 흐름으로 먼저 해결하고, 커뮤니티 / 랭킹 / 홈바리스타 확장 기능을 1차로 연결하는 데 집중한다.

## 현재 MVP 범위

1. 이메일 로그인
2. 지도 기반 카페 탐색
3. 카페 상세 정보
4. 리뷰 작성 및 리뷰 목록 조회
5. 카페 저장(북마크)
6. 마이페이지 활동 요약
7. 커뮤니티 게시판
8. 카페 랭킹
9. 홈바리스타 레시피 피드
10. Supabase 기반 카페 / 리뷰 / 북마크 / 커뮤니티 / 홈바리스타 데이터 구조

## 현재 로컬에서 완료된 것

1. iOS SwiftUI 앱 골격
2. `XcodeGen` 기반 프로젝트와 `ios/BrewSpot.xcodeproj`
3. Supabase Auth / DB 연결
4. Email 로그인 UI 연결
5. 카페 목록 / 상세 / 리뷰 / 저장 / 마이페이지 화면 구현
6. 커뮤니티 / 랭킹 / 홈바리스타 1차 화면 구현
7. 카페 24개 / 리뷰 36개 시드 초안 작성
8. 운영 정책, 개인정보처리방침, 이용약관 기준안 정리
9. 시뮬레이터 빌드 확인
