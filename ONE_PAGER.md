# BrewSpot 1페이지 요약

최종 기준일: 2026-04-24

## 프로젝트

`BrewSpot`  
내 취향에 맞는 카페를 찾고, 저장하고, 리뷰로 기록하는 iOS 커피 앱

## 지금 풀고 있는 문제

카페 정보는 많지만 사용자는 여전히 "내 취향에 맞는 카페"를 빠르게 찾기 어렵다.  
현재 MVP는 이 문제를 `지도 + 카페 상세 + 리뷰 + 저장 + 마이페이지 기록` 흐름으로 먼저 해결하는 데 집중한다.

## 현재 MVP 범위

1. 이메일 / Google / Apple 로그인
2. 지도 기반 카페 탐색
3. 카페 상세 정보
4. 리뷰 작성 및 리뷰 목록 조회
5. 카페 저장(북마크)
6. 마이페이지 활동 요약
7. Supabase 기반 카페 / 리뷰 / 북마크 데이터 구조

## 현재 로컬에서 완료된 것

1. iOS SwiftUI 앱 골격
2. `XcodeGen` 기반 프로젝트와 `ios/BrewSpot.xcodeproj`
3. Supabase Auth / DB 연결
4. Google / Apple / Email 로그인 UI 연결
5. 카페 목록 / 상세 / 리뷰 / 저장 / 마이페이지 화면 구현
6. 카페 24개 / 리뷰 36개 시드 초안 작성
7. 운영 정책, 개인정보처리방침, 이용약관 기준안 정리
8. 시뮬레이터에서 앱 실행 확인
