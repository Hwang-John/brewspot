## 1. 사전 조건

- [ ] 최신 앱 빌드 설치

## 2. 이메일 로그인 기본 흐름

- [ ] 회원가입 화면 진입
- [ ] 신규 이메일로 회원가입 시도
- [ ] 성공 또는 실패 메시지가 이해 가능한 문구로 표시되는지 확인
- [ ] 이메일 확인이 필요한 경우 안내 문구 확인
- [ ] 기존 계정으로 로그인
- [ ] 로그인 후 홈 화면으로 이동하는지 확인

정상 기준:
- 로그인 성공 후 홈 탭이 보인다
- 에러 발생 시 원인을 추측할 수 있는 문구가 보인다

## 3. 카페 조회 / 탐색

- [ ] 홈 목록에 카페가 0개가 아닌 상태로 보이는지 확인
- [ ] 검색어 입력 시 필터링이 동작하는지 확인
- [ ] 카테고리 선택 시 목록이 변하는지 확인
- [ ] 지도 화면이 열리고 카페 좌표가 보이는지 확인
- [ ] 카페 상세 진입이 가능한지 확인

정상 기준:
- 빈 화면이 아니라 실제 카페 데이터가 보인다
- 목록, 지도, 상세 간 이동이 끊기지 않는다

## 4. 리뷰 작성 / 반영

- [ ] 카페 상세에서 리뷰 작성 시트 열기
- [ ] 평점, 내용, 추천 메뉴 입력 후 저장
- [ ] 저장 직후 상세 화면 리뷰 목록에 반영되는지 확인
- [ ] 마이페이지 내 리뷰 목록에 반영되는지 확인
- [ ] 최근 활동 섹션에도 반영되는지 확인

정상 기준:
- 저장 후 중복 크래시 없이 화면이 유지된다
- 같은 리뷰가 상세 / 마이페이지 / 최근 활동에서 일관되게 보인다
