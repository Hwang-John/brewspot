# BrewSpot iOS Scaffold

현재 폴더는 BrewSpot iOS 앱의 초기 SwiftUI 골격이다.

포함된 것:

1. Supabase 연결 설정
2. Email 로그인
3. 회원가입
4. 세션 확인
5. 로그아웃
6. 로그인 후 임시 홈 화면

## 중요

이 환경에서는 Xcode 빌드를 실행할 수 없어서, 실제 실행 확인은 Mac + Xcode에서 해야 한다.

## 권장 실행 순서

1. Mac에서 이 저장소 열기
2. `XcodeGen` 설치
3. `ios/project.yml` 기반으로 프로젝트 생성
4. Supabase Swift 패키지 설치 확인
5. 실행
