# BrewSpot App Review 메모 초안

최종 기준일: 2026-04-24

이 문서는 App Store 심사 제출 시 `App Review Notes`에 넣을 내용을 미리 정리한 초안이다.

## 1. 서비스 요약

BrewSpot은 카페를 탐색하고, 저장하고, 리뷰를 남길 수 있는 iOS 앱입니다.  
현재 버전은 지도 기반 탐색, 카페 상세, 리뷰 작성, 북마크, 마이페이지 기능에 집중한 MVP입니다.

## 2. 로그인 관련 메모 초안

- 기본 로그인 수단: Email
- 추가 로그인 수단: Google, Apple
- 심사 전 확인 필요: Google / Apple Provider가 실제로 활성화되어 있는지 확인

리뷰어용 계정 입력 자리:
- Email: `[입력 필요]`
- Password: `[입력 필요]`

## 3. 데이터 관련 메모 초안

- 심사용 계정은 시드된 카페와 리뷰 데이터를 볼 수 있어야 함
- 초기 데이터 기준:
  - 카페 24개
  - 리뷰 36개
- 저장과 마이페이지 흐름 확인을 위해 테스트 계정에 일부 활동 데이터가 준비되어 있어야 함

## 4. 권한 / 개인정보 관련 메모 초안

- 현재 버전은 사용자 현재 위치를 수집하지 않음
- 지도는 등록된 카페 좌표 표시용으로만 사용됨
- 사진 업로드, 연락처 접근, 알림 권한 요청 없음

## 5. 심사 전 최종 체크

- [ ] 리뷰어용 테스트 계정 입력
- [ ] 로그인 가능한 Provider 상태 확인
- [ ] 개인정보처리방침 URL 입력
- [ ] 이용약관 URL 입력
- [ ] 주요 기능이 실제 시드 데이터와 함께 보이는지 확인
