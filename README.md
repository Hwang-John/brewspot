# BrewSpot PRD

최종 기준일: 2026-05-01

## 빠른 소개

이 저장소는 `BrewSpot` iOS 커피 앱의 현재 MVP 기준 문서와 운영 준비 문서를 정리한 저장소다.

현재 제품 방향:

1. 지도 기반 카페 탐색
2. 카페 상세 정보
3. 리뷰 작성과 조회
4. 카페 저장
5. 마이페이지 활동 요약
6. 커뮤니티 게시판
7. 카페 랭킹
8. 홈바리스타 피드

## 현재 기준 문서

1. `TODO.md`: 지금 해야 할 일과 완료된 작업
2. `ONE_PAGER.md`: 1페이지 요약

## 1. 제품 한 줄 정의

`내 취향에 맞는 카페를 찾고, 저장하고, 리뷰로 기록하는 iOS 커피 앱`

## 2. 현재 MVP 범위

### 현재 구현 완료

1. 이메일 로그인 / 회원가입 UI 연결
2. 홈 검색 / 카테고리 필터
3. 지도 기반 카페 탐색
4. 카페 상세
5. 리뷰 작성 및 리뷰 목록
6. 북마크 저장
7. 마이페이지 활동 요약
8. 커뮤니티 게시판 탭 / 목록 / 상세 / 글쓰기
9. 랭킹 탭과 온디바이스 랭킹 계산
10. 홈바리스타 탭 / 목록 / 상세 / 글쓰기
11. Supabase 기반 카페 / 리뷰 / 북마크 / 커뮤니티 / 홈바리스타 서비스

### 현재 미구현 또는 외부 검증 필요

1. Supabase 실스키마 반영
2. 카페 24개 / 리뷰 36개 실제 입력
3. end-to-end 동작 검증
